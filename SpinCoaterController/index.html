Profile Editor Guide

Create multi-step spin recipes.

Step Parameters

  • RPM: Target speed for this step.
  • Ramp (ms): Time taken to reach the target RPM from the previous speed.
  • Hold (ms): Time to maintain the target RPM before moving to the next step.

Ramp Types

  • Linear (Lin): Constant acceleration. Standard for most uses.
  • Exponential (Exp): Starts slow, accelerates fast. Good for delicate films.
  • S-Curve (S-C): Smooth start and stop. Reduces mechanical jerk. Best for high speeds.

PID Tuning Guide

Low Pass Filter (Alpha)

Filters sensor noise. Range 0.0 - 1.0.

  • 1.0: No filtering (Raw).
  • 0.1: Heavy filtering (Slow response).
  • Recommended: 0.3 - 0.8 depending on noise.

Integral Windup Range

Prevents overshoot during ramp-up.

  • Defines RPM error range where 'I' term is active.
  • Too Small: Target never reached.
  • Too Large: Overshoot.
  • Recommended: 300-500 RPM.

PID Constants

Kp (Proportional): Immediate reaction force. Increase until oscillation starts.

Ki (Integral): Fixes steady-state error. Increase to reach exact target.

Kd (Derivative): Dampens oscillation. Increase to reduce overshoot.

ESC Calibration Guide

What is it?

ESC Calibration synchronizes the controller's throttle signal (1000-2000us) with the ESC's input range.

When to perform it?

  • First Setup: Essential for new builds.
  • Deadband Issues: If motor doesn't spin until high throttle.
  • Top Speed: If motor reaches max speed too early or never reaches it.
  • New Hardware: After replacing ESC or Arduino.

Process

The wizard sends MAX throttle (2000us) on boot to enter calibration mode, then MIN throttle (1000us) to save the range.

ESC Calibration

WARNING:Remove samples from spin coater if possible.

1. Disconnect ESC Power (Battery/Plug).

2. Keep USB connected to Arduino.

3. Click Start to send MAX throttle.

Sending MAX Throttle (2000us)

1. Connect ESC Power NOW.

2. Wait for "Musical Tone" then "2 Beeps".

3. Immediately click Next.

Sending MIN Throttle (1000us)

1. Wait for "Arming Beeps".

2. Calibration Complete.
